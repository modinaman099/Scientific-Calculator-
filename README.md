
# Advanced Scientific Calculator

A modern scientific calculator built using HTML, CSS, and JavaScript. It combines everyday calculations with advanced mathematical functions, a live currency converter, memory operations, calculation history, and a clean responsive interface.

The goal of this project was to create a calculator that feels complete rather than just solving basic arithmetic. It includes features that are useful for students, professionals, and anyone who regularly works with calculations.

---

## Features

### Basic Calculator
- Addition, subtraction, multiplication, and division
- Decimal calculations
- Percentage support
- Positive/negative toggle
- Backspace and clear functions

### Scientific Functions
- Square and cube
- Square root and cube root
- Power calculations
- Nth root
- Reciprocal
- Factorial
- Random number generator
- Trigonometric functions
  - sin
  - cos
  - tan
  - asin
  - acos
  - atan
- Logarithms
  - log₂
  - log₁₀
  - natural log
  - custom base logarithm
- Pi and Euler's constant
- Degree/Radian mode

### Memory Operations
- Memory Store
- Memory Recall
- Memory Clear
- Memory Add
- Memory Subtract

### Currency Converter
- Live exchange rates
- Supports INR, USD, EUR, and GBP
- Swap currencies instantly
- Automatic fallback rates if the live API is unavailable

### Additional Features
- Calculation history
- Copy result to clipboard
- Previous answer (Ans)
- Keyboard support
- Light and dark themes
- Advanced mode toggle
- Responsive layout for desktop and mobile devices

---

## Technologies Used

- HTML5
- CSS3
- Vanilla JavaScript
- ExchangeRate API for live currency conversion

No frameworks or external libraries were used for the calculator itself.
